Bot Commands

Pan-pan's command prefix is panpan. There's multiple commands available for now, such as:

  • panpan help
    Command list
  • panpan date
    Send cuurent time in GMT+7
  • panpan time
    Send today's date in GMT+7
  • panpan genshinch <character-name>
    Provide a minimalized info about a Genshin character.

and more coming soon!
